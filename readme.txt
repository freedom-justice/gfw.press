
# fork自[GFW.Press](https://github.com/chinashiyu/gfw.press)。是PC端程序。


### 如果需要编译运行，那么请注意

> 1.此项目需要使用jdk8才能编译运行

> 2.需要将[jce_policy-8.zip](https://github.com/lyx32/gfw.press/blob/master/jce_policy-8.zip)中的jar拷贝到$JAVA_HOME\jre\lib\security中并覆盖原文件

> 3.根目录中[jssecacerts](https://github.com/lyx32/gfw.press/blob/master/jssecacerts)用于自动获取节点信息
